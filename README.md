# Conglomerate Gaze Games

Two gaze-controlled games built in Processing with an integrated launcher.

## Games

- **Water the Plants** - Hover over plants to water them and watch flowers bloom
- **Memory Match** - Match pairs of cards by hovering to flip them

## How to Run

Open `launcher/conglomerate.pde` in Processing and run.

Hover over a game card for 3 seconds to launch it. Press ESC to return to the menu.
